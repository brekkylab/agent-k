# 마케팅 지표

담당: 마케팅
목적: 퍼널과 채널 성과 추적

이 문서가 다루는 건 퍼널 지표다. 방문 수, 리드 수, 전환율 같은 것들이고 매출이 아니다. 여기 나오는 백분율과 숫자는 수요를 얼마나 만들어냈는지를 설명하지 우리가 벌어들인 돈을 말하는 게 아니다.

## 퍼널 (직전 기간)

윗단에서 아랫단으로 내려갈수록 숫자가 좁혀지는 전형적인 깔때기 모양이다.

| 단계 | 값 |
|------|-----|
| 웹사이트 방문 | 240,000 |
| 마케팅 적격 리드(MQL) | 4,800 |
| 영업 적격 리드(SQL) | 1,150 |
| 수주 성사(Closed-won) | 410 |
| 방문 → MQL | 2.0% |
| MQL → SQL | 24% |
| SQL → 성사 | 36% |

## 채널

오가닉 검색이 MQL을 가장 많이 물어다 준다. 콘텐츠와 증거 자료는 후반 단계, 그러니까 SQL에서 성사로 넘어가는 전환율이 제일 좋다. 유료 채널은 SMB에는 효율적인데 엔터프라이즈에는 힘을 못 쓴다.

## 잘되는 것

증거 기반 콘텐츠가 후반 단계를 잘 전환시키고 있다. 웨비나는 엔터프라이즈 파이프라인을 꾸준히 끌어온다. 둘 다 투입 대비 효과가 분명한 채널이다.

## 안되는 것

유료 소셜은 엔터프라이즈 쪽에서 계속 부진하다. 상단 퍼널을 노린 일반 콘텐츠는 트래픽은 모으지만 의도가 낮은 리드를 데려와서 아랫단 전환으로 이어지지 않는다.

## 계획

증거 자료에 더 집중한다. 유료 예산은 새 고객을 무차별로 긁기보다 이미 관심을 보인 따뜻한 어카운트를 리타게팅하는 쪽으로 옮긴다. 엔터프라이즈를 위해서는 통합에 드는 부담을 한 장으로 정리한 원페이저를 만든다.

## 메모

지표는 따로 표기하지 않으면 세그먼트가 섞인 혼합값이다. 어트리뷰션은 라스트터치 기준이라 방향성 정도로만 받아들여야 한다.

## 어느 문서를 봐야 하나

여기 어떤 숫자도 매출이 아니다. 작년 매출이 궁금하면 FY2025 연차 보고서를 봐야 한다. 이 파일은 오직 수요 창출만 추적한다.

## 주기

마케팅은 매주 이 지표를 들여다보고, 리더십은 월간 롤업으로 받아 본다.
